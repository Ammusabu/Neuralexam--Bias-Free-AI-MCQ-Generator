# ⚡ NeuralExam – Bias-Free AI MCQ Generator

NeuralExam is an AI-powered MCQ generation platform designed to create **fair, unbiased, and balanced assessments** from any input source.

Unlike typical quiz generators, NeuralExam ensures that answer distributions are **evenly balanced (A/B/C/D)** to prevent pattern-based guessing and bias.

---

## 🌐 Live Demo

👉 https://neuralexam.netlify.app

---

## 🧠 Key Features

### 🎯 Bias-Free Question Generation

* Automatically balances correct answers across A/B/C/D
* Prevents predictable answer patterns
* Ensures fair evaluation for all users

### 🤖 AI-Powered MCQs

* Uses LLM (Groq + LLaMA 3.1)
* Generates high-quality questions with explanations
* Supports multiple difficulty levels (Easy, Medium, Hard)

### 📥 Multiple Input Sources

* Topic-based generation
* Paste custom text
* Upload PDF documents

### 🔄 Dual Modes

* 🔒 **Exam Mode** – Hidden answers for real assessment
* 📖 **Practice Mode** – Instant feedback with explanations

### 📊 Performance Analytics

* Score calculation with percentage
* Grade classification
* Question-wise explanations
* Answer tracking system

### ⚡ Real-Time UI

* Interactive quiz interface
* Progress tracking bar
* Smooth UX with modern design

* Frontend: Netlify

---

## ⚙️ How It Works

1. User provides input (topic/text/PDF)
2. Backend generates MCQs using LLM
3. System applies:

   * Bias correction (balanced answers)
   * Option normalization
4. Frontend renders quiz dynamically
5. User submits answers → results calculated

---

## 🔥 Unique Feature: Bias Correction Engine

Traditional MCQ generators often:

* Repeat answer patterns (e.g., mostly "C")
* Create unfair assessments

NeuralExam solves this by:

* Distributing correct answers evenly
* Randomizing options intelligently
* Ensuring no answer stands out by length
